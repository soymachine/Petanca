0. Buscar imagen en internet (Google Images)

Usa la Google Custom Search JSON API (gratis hasta 100 búsquedas/día): crea una clave de API en Google Cloud Console y un motor en Programmable Search Engine con "Búsqueda de imágenes" activada y "Buscar en toda la web" activado. Se guardan solo en este navegador (localStorage), nunca se envían a ningún sitio nuestro.

Arrastra una imagen aquí, o haz clic para elegir
JPG/PNG, idealmente un retrato recortado a cara/hombros — o elige un resultado de la búsqueda de arriba
Nombre (opcional)

Es solo una etiqueta interna del fichero (el jugador nunca la ve — su nombre lo genera el juego aparte). Si lo dejas vacío se asigna uno tipo POOL-N; si lo rellenas, se comprueba que no exista ya.

Amuleto a sustituir

Son 5 fijos, no se pueden añadir más. "+ Añadir" reemplaza el icono de este amuleto por la foto de arriba — el anterior se pierde.

Tamaño (cols × rows)

60×39 / 18×12 son los tamaños estándar del pool actual. Para RIVAL_FACES solo se usa la mini, guardada como "photo" (18×12) — así lo espera ese fichero.
